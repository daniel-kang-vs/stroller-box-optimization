The Problem

Why does box selection matter for shipping?

Dimensional Weight

Carriers charge by dimensional weight (L×W×H / 250) when it exceeds actual weight. Oversized boxes inflate shipping costs even for light products.

Box Standardization

Using a unique box per product is impractical. Warehouses need a limited set of box sizes that can cover all products efficiently.

The Trade-off

Fewer box types = simpler operations, but potentially higher shipping cost. The model finds the sweet spot that minimizes cost with at most 7 box types.

Model Formulation

Integer Linear Program solved with Gurobi optimizer

Decision Variables

xij ∈ {0, 1} — assign stroller j to box i

yi ∈ {0, 1} — use box type i in the system

Objective

Minimize Σi,j costij · xij

where cost = max(dimensional weight, actual weight)

Constraints

1. Each stroller assigned exactly one box

2. Assignment only if box type is selected

3. Stroller must physically fit in the box

4. At most 7 box types total

Fit Criteria

A stroller fits a box when all three dimensions (with 0.5″ padding) are ≤ the corresponding box dimensions.

Only 731 of 731 possible pairs actually fit

Key Takeaways

  • S-16765 (36×24×18) is the workhorse box, covering 5 of 17 products efficiently.
  • Vista V2 and Fox 5 are the costliest to ship — their bulky dimensions force the largest box (40×40×40), driving 30% of total shipping weight.
  • The 7-box system adds only 6.5% extra shipping weight compared to a theoretically perfect per-product match, a very acceptable trade-off for warehouse simplicity.
  • Most of the penalty comes from just 2 products (Vista V2, Fox 5) needing oversized boxes; the other 15 products are packed near-optimally.
